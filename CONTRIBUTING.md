### Reporting Bugs

If you found a bug:

1. Check if it hasn't already been reported in [Issues](https://github.com/sh1dan/infoseek/issues)
2. Create a new issue describing the problem
3. Include steps to reproduce the bug
4. Attach screenshots if possible

### Suggesting Enhancements

To suggest a new feature:

1. Create an issue describing the proposed feature
2. Explain why this feature would be useful
3. Describe how it should work

### Pull Request

1. Fork the repository
2. Create a branch for your feature (`git checkout -b feature/AmazingFeature`)
3. Make your changes
4. Ensure code follows the project's style guidelines
5. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
6. Push to the branch (`git push origin feature/AmazingFeature`)
7. Open a Pull Request
